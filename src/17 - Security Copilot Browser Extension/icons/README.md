# Icon Placeholder

The extension requires PNG icons in the following sizes:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Creating Icons

You can create these icons using:

1. **Online Tools**:
   - [Canva](https://www.canva.com/) - Free design tool
   - [Figma](https://www.figma.com/) - Design tool with export options
   - [GIMP](https://www.gimp.org/) - Free image editor

2. **Icon Libraries**:
   - [Icons8](https://icons8.com/)
   - [Flaticon](https://www.flaticon.com/)
   - [Font Awesome](https://fontawesome.com/)

3. **Design Guidelines**:
   - Use a shield or security-themed icon
   - Keep it simple and recognizable at small sizes
   - Use the Security Copilot blue color (#0078d4)
   - Ensure sufficient contrast

4. **Quick Solution**:
   Use the provided `icon.svg` and convert it to PNG at different sizes:
   - Open icon.svg in a browser
   - Take screenshots at different zoom levels
   - Resize to exact dimensions needed
   - Save as PNG files

## Temporary Workaround

If you want to test the extension without custom icons, you can:
1. Use any PNG image
2. Rename it to the required filenames
3. The extension will work, just with generic icons

Remember to replace these placeholder instructions with actual PNG files before publishing the extension!
